# 🌦️ Weather App

A simple and beautiful **Weather Web App** that lets you check real-time weather conditions of any city in the world. Built with modern web technologies and a clean UI for a smooth user experience.

---

## ✨ Features

🌍 Search weather for any city
🌡️ Real-time temperature updates
☁️ Weather conditions (Cloudy, Sunny, Rainy, etc.)
💨 Wind speed and humidity info
📱 Responsive design for all devices

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 CSS
* 🌐 OpenWeatherMap API
* ⚡ JavaScript (ES6)

---

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/anmolsin1124/weatherpro.git
```

### 2️⃣ Go to the project folder

```bash
cd weatherpro
```

### 3️⃣ Configure your API key

Create a `.env` file in the project root (copy from `.env.example`):

```bash
cp .env.example .env
```

Then edit `.env` and add your [OpenWeatherMap API key](https://openweathermap.org/api):

```
VITE_WEATHER_API_KEY=your_api_key_here
```

### 4️⃣ Install dependencies

```bash
npm install
```

### 5️⃣ Start the development server

```bash
npm run dev
```

Now open **http://localhost:5173** in your browser.
